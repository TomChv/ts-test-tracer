# TypeScript test tracer

Utility libraries to automatically trace tests inside `dagger run`.

### Integration

- [x] Bun ([`@otel-test-runner/bun-test`](https://www.npmjs.com/package/@otel-test-runner/bun-test))
- [x] Jest ([`@otel-test-runner/jest-test`](https://www.npmjs.com/package/@otel-test-runner/jest-test))
- [x] Mocha ([`@otel-test-runner/mocha-test`](https://www.npmjs.com/package/@otel-test-runner/mocha-test))
- [ ] Deno
